# Jezreel Portfolio Website

A personal portfolio website built with plain HTML, CSS, and JavaScript.

## Overview

This project includes a redesigned landing page inspired by a monochrome, editorial portfolio style and supporting pages for:

- Skills
- Education
- Blog (starter layout)
- Contact

## Project Structure

- `Portfolio.html` - Main landing page
- `skills.html` - Skills showcase
- `education.html` - Education section
- `blog.html` - Blog starter page
- `contact.html` - Contact form page
- `style.css` - Main styling and layout system
- `style_additional.css` - Additional page-specific styles
- `script1.js` - Typing + smooth-scroll behavior
- `script2.js` - Navigation/page transition helpers

## How to Run

Because this is a static project, you can run it directly:

1. Clone/download the repository.
2. Open `Portfolio.html` in a browser.

For better local development, serve the folder with a local static server (for example, VS Code Live Server).
